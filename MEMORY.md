## 工作原则（不可违背）

### 1. 生成后必检查原则
**每次生成网页/报告后，必须立即检查结果，确保内容正确、完整、无错误。**

检查清单：
- [ ] 所有页面是否正常生成（数量正确）
- [ ] 页面内容是否正确填充（非空、非模板文本）
- [ ] 链接是否可正常跳转
- [ ] 样式是否正常显示
- [ ] 数据是否真实有效

**发现问题立即修复，不提交有问题的版本。**

### 2. 质量验证流程
1. 生成 → 2. 自检 → 3. 修复 → 4. 再检 → 5. 提交

### 3. 用户反馈优先
- 用户指出的问题优先处理
- 不找借口，立即检查修复
- 修复后主动验证
