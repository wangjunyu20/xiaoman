# Skill 质量检查清单

> 版本: v1.0  
> 适用: 投资研究群空间所有 Skill  
> 更新日期: 2026-03-09

## 一、文档完整性检查

### 1.1 必需文件
- [ ] `SKILL.md` - 主文档存在且格式正确
- [ ] `scripts/` - 脚本目录存在（如适用）
- [ ] `references/` - 参考资料目录存在（推荐）
- [ ] `README.md` - 简要说明（可选但推荐）

### 1.2 SKILL.md 结构检查
- [ ] **元数据头** - 包含规范的 YAML frontmatter
  ```yaml
  ---
  name: skill-name
  description: 清晰描述用途
  allowed-tools:
    - Bash(python:*)
    - Read
  ---
  ```
- [ ] **快速开始** - 有独立的 Quick Start 章节
- [ ] **执行流程** - 清晰的步骤说明
- [ ] **故障排查** - 常见问题及解决方案
- [ ] **示例代码** - 至少1个可运行的示例

### 1.3 文档质量标准
- [ ] 语言一致（中文/英文统一）
- [ ] 格式规范（Markdown 语法正确）
- [ ] 链接有效（内部引用可访问）
- [ ] 版本标注（更新日期/版本号）

## 七、检查工具

### 7.1 自动化检查脚本
```bash
# 运行所有检查
./scripts/check_setup.sh

# 检查特定 Skill
./scripts/check_setup.sh tushare-finance
```

### 7.2 手动检查流程
1. 阅读 SKILL.md，确认文档完整
2. 运行示例代码，确认功能正常
3. 检查错误处理，确认提示友好
4. 验证输出格式，确认符合规范

## 八、评分标准

| 维度 | 权重 | 评分 |
|------|------|------|
| 文档完整性 | 30% | 0-10 |

| 安全可维护 | 20% | 0-10 |

- 9-10: 优秀 ✓
