# 📊 行业研究报告模板

> 版本：v1.0 | 创建日期：2026-03-07
> 目标：确保每份报告对投资有实际帮助

---

## 模板结构

```
一、行业概览
二、本周重大资讯（必须含日期）
三、A股对应板块表现
四、美股对标公司对比（必须含昨日走势）
五、行业趋势分析
六、投资风险提示
七、数据来源与校验
```

---

## 章节详细规范

### 一、行业概览

**目标**：让读者快速了解行业当前状态

**格式**：
```
## 一、行业概览

- 行业名称：[名称]
- 报告日期：[YYYY-MM-DD]
- 本周关键词：[3-5个核心词]
```

**数据来源**：自己整理

---

### 二、本周重大资讯 ⚠️ 重点章节

**目标**：提供最新、最重要的新闻，对投资决策有帮助

**格式**：
```
## 二、本周重大资讯

### 2.1 政策动态
| 日期 | 来源 | 资讯标题 | 投资影响 |
|------|------|----------|----------|
| 2026-03-07 | 新华社 | xxx | 利好/利空/中性 |

### 2.2 技术突破
| 日期 | 来源 | 突破内容 | 投资影响 |
|------|------|----------|----------|

### 动态
| 日期2.3 公司 | 来源 | 公司 | 事件 | 投资影响 |
|------|------|------|------|----------|
```

**必填字段**：
- **日期**（YYYY-MM-DD）：必须精确到日，严禁使用"近日""最近"
- **来源**：必须标注媒体/机构名称
- **投资影响**：利好/利空/中性 + 简要说明

**数据来源与搜索规则**：
1. 搜索 `[行业名] + 新闻 + 昨日/今日`
2. 优先来源：财联社、同花顺、新华网、澎湃新闻
3. 搜索英文：`[industry] + breaking news + today`

**校验机制**：
- [ ] 每条资讯必须有明确日期
- [ ] 无法确定日期的资讯标记为"时间待确认"
- [ ] 同一事件多源交叉验证

---

### 三、A股对应板块表现

**目标**：了解A股相关板块本周走势

**格式**：
```
## 三、A股对应板块表现

- 同花顺行业指数：[代码] 
- 本周涨跌幅：[+X.X%]
- 领涨/领跌个股：
  - 领涨：[代码] [名称] [+X%]
  - 领跌：[代码] [名称] [-X%]
```

**数据来源**：
- 同花顺APP/官网 → 行业板块 → 具体行业
- 东方财富网 → 行业板块

**校验机制**：
- [ ] 涨跌幅数据需与当日收盘价核对
- [ ] 领涨领跌个股需有具体涨跌幅数字

---

### 四、美股对标公司对比 ⚠️ 重点章节

**目标**：通过美股走势预判A股开盘走势（时差因素）

**格式**：
```
## 四、美股对标公司对比

### 4.1 对标公司昨日走势
| 公司 | 代码 | 昨日收盘价 | 涨跌幅 | 近期趋势 |
|------|------|-----------|--------|----------|
| QuantumScape | QS | $X.XX | +X.XX% | 上升/下降/震荡 |

### 4.2 今日盘前表现（如果有）
| 公司 | 代码 | 盘前涨跌幅 | 原因 |
|------|------|-----------|------|

### 4.3 对A股影响分析
[分析内容]
```

**必填字段**：
- **昨日收盘价**：精确到小数点后两位
- **涨跌幅**：精确到百分比（如 +5.23%）
- **数据时间**：标注"昨日"具体日期

**数据来源**：
1. Yahoo Finance（yfinance库）
2. Google Finance
3. 雪球/富途 盘前数据

**搜索规则**：
- 搜索 `[股票代码] + stock price + yesterday`
- 搜索 `[公司名] + stock + today + premarket`

**校验机制**：
- [ ] 股价必须为真实数值（不可估算）
- [ ] 涨跌幅可通过 `(收盘价-昨日收盘价)/昨日收盘价` 验算
- [ ] 注明数据获取时间

---

### 五、行业趋势分析

**目标**：基于资讯和数据，给出行业走向判断

**格式**：
```
## 五、行业趋势分析

### 5.1 短期趋势（1-4周）
- 驱动因素：[因素1]、[因素2]
- 趋势判断：[上升/下降/震荡]
- 置信度：[高/中/低]

### 5.2 中期趋势（1-6个月）
[同上格式]

### 5.3 长期趋势（6个月以上）
[同上格式]
```

**要求**：
- 结论必须有数据/资讯支撑
- 标注置信度（高/中/低）

---

### 六、投资风险提示

**目标**：提示潜在风险

**格式**：
```
## 六、投资风险提示

- 风险1：[描述] | 概率：[高/中/低] | 影响：[大/中/小]
- 风险2：[描述] | 概率：[高/中/低] | 影响：[大/中/小]
```

### 七、数据来源与校验

**目标**：透明化数据来源，便于核查

**格式**：
```
## 七、数据来源与校验

### 7.1 数据来源
| 数据类型 | 来源 | 获取时间 | 状态 |
|----------|------|-----------|------|
| 行业资讯 | 财联社 | 2026-03-07 19:30 | ✅ 已验证 |
| 美股行情 | Yahoo Finance | 2026-03-07 19:30 | ✅ 已验证 |

### 7.2 校验记录
- [ ] 资讯日期已确认
- [ ] 股价已与至少2个来源交叉验证
- [ ] 代码与名称匹配已确认
```
