# 多源新闻数据聚合方案

## 需求背景
当前行业研究报告新闻资讯不够丰富，需要增加权威财经新闻源。

## 新增数据源

| 序号 | 新闻源 | 网址 | 数据类型 | 优先级 |
|------|--------|------|----------|--------|
| 1 | 新华网财经 | news.xinhuanet.com | 政策/宏观新闻 | P0 |
| 2 | 新浪财经 | finance.sina.com.cn | 综合财经新闻 | P0 |
| 3 | 东方财富股吧 | guba.eastmoney.com | 个股/板块舆情 | P0 |
| 4 | 财联社 | cls.cn | 实时快讯/电报 | P0 |
| 5 | 中国基金报 | chinafundnews.com | 基金/机构观点 | P1 |

## 技术方案

### 1. 架构设计
```
news_fetcher.py (主模块)
├── fetch_xinhua(keyword)      # 新华网
├── fetch_sina_finance(keyword) # 新浪财经
├── fetch_eastmoney_guba(code)  # 东财股吧
├── fetch_cls_news(keyword)     # 财联社
└── fetch_chinafund(keyword)    # 中国基金报
```

### 2. 按行业映射关键词

| 行业 | 搜索关键词 |
|------|-----------|
| 光模块 | 光模块、CPO、光通信、中际旭创 |
| 人形机器人 | 人形机器人、机器人、优必选、特斯拉机器人 |
| 量子科技 | 量子科技、量子计算、量子通信、国盾量子 |
| AI应用 | AI应用、人工智能、大模型、AIGC |
| 可控核聚变 | 核聚变、人造太阳、ITER、东方电气 |
| 脑机接口 | 脑机接口、Neuralink、三博脑科 |
| AI+医疗 | AI医疗、智慧医疗、医疗AI、卫宁健康 |

### 3. 数据输出格式
```json
{
  "source": "财联社",
  "title": "标题",
  "url": "链接",
  "publish_time": "2026-03-08 09:30:00",
  "summary": "摘要",
  "industry": "光模块",
  "impact": "利好/利空/中性"
}
```

### 4. 整合到报告生成流程

修改 `generate_reports.py`:
1. 生成报告前调用 `news_fetcher.fetch_all(industry)`
2. 将新闻数据注入行业卡片模板
3. 增加"最新资讯"栏目

## 注意事项

1. **反爬策略**：各网站都有反爬机制，需要设置合理请求间隔
2. **数据缓存**：新闻数据可缓存1-2小时，避免频繁请求
3. **失败降级**：某个源失败时，自动切换到其他源
4. **合规性**：仅抓取公开可访问的新闻标题和摘要，不抓取全文
