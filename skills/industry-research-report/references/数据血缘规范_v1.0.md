# 数据血缘规范 v1.0

> 建立数据"输入=输出"的勾稽关系，确保数据可追溯、可验证

## 核心目标

1. **可追溯**：知道每条数据从哪来、经过哪些处理
2. **可验证**：能核对"采集40条 → 展示15条"是否合理
3. **可审计**：发现问题时能定位是哪个环节出错

## 代码实现

### 在news_fetcher中增加日志

```python
class NewsFetcher:
    def __init__(self):
        self.lineage_log = {
            "collection": {},
            "cleaning": {},
            "labeling": {}
        }
    
    def fetch_source(self, source_name, industry):
        """抓取单个数据源，记录血缘"""
        news_list = self._do_fetch(source_name, industry)
        
        # 记录采集层血缘
        self.lineage_log["collection"][source_name] = {
            "count": len(news_list),
            "timestamp": datetime.now().isoformat()
        }
        
        return news_list
    
    def clean_news(self, all_news):
        """清洗新闻，记录血缘"""
        input_count = len(all_news)
        
        # 执行清洗
        valid_news = [n for n in all_news if self._is_valid_url(n['url'])]
        invalid_removed = input_count - len(valid_news)
        
        unique_news = self._remove_duplicates(valid_news)
        duplicate_removed = len(valid_news) - len(unique_news)
        
        # 记录清洗层血缘
        self.lineage_log["cleaning"] = {
            "input": input_count,
            "invalid_url_removed": invalid_removed,
            "duplicate_removed": duplicate_removed,
            "output": len(unique_news)
        }
        
        return unique_news
```

### 在generate_reports中验证勾稽

```python
def verify_lineage(industry, lineage_log):
    """验证数据勾稽关系"""
    collection_total = sum(
        s["count"] for s in lineage_log["collection"].values()
    )
    cleaning_input = lineage_log["cleaning"]["input"]
    cleaning_output = lineage_log["cleaning"]["output"]
    labeling_input = lineage_log["labeling"]["input"]
    
    # 检查1: 采集总数 = 清洗输入
    assert collection_total == cleaning_input, \
        f"{industry}: 采集{collection_total} ≠ 清洗输入{cleaning_input}"
    
    # 检查2: 清洗输入 - 过滤 = 清洗输出
    expected_output = (
        cleaning_input 
        - lineage_log["cleaning"]["invalid_url_removed"]
        - lineage_log["cleaning"]["duplicate_removed"]
    )
    assert expected_output == cleaning_output, \
        f"{industry}: 清洗计算错误"
    
    # 检查3: 清洗输出 = 标注输入
    assert cleaning_output == labeling_input, \
        f"{industry}: 清洗输出 ≠ 标注输入"
    
    return True
```

## 故障排查

### 勾稽关系错误示例

**错误**：`采集40 ≠ 清洗输入35`

**排查步骤**：
1. 检查是否有数据源未记录到collection
2. 检查清洗前是否有预处理删除了数据
3. 检查日志记录是否完整

**错误**：`清洗输出32 ≠ 标注输入30`

**排查步骤**：
1. 检查清洗后是否有额外的过滤逻辑
2. 检查数据是否在传递过程中丢失
3. 检查异步处理是否有race condition
