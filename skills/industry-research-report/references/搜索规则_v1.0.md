## 搜索命令模板

### 1. 中文行业重大新闻搜索

**搜索公式**：
```
[行业名] + [新闻/资讯/突破/动态] + [时间范围]
```

**示例**：
```
量子科技 + 重大突破 + 2026年3月
量子科技 + 政策 + 最近一周
量子科技 + 投资 + 2026-03
```

## 搜索时间规则

### 搜索范围定义

| 关键词 | 含义 | 实际搜索范围 |
|--------|------|-------------|
| 今日 | 今天 | 2026-03-07 |
| 昨日 | 昨天 | 2026-03-06 |
| 本周 | 周一至今 | 2026-03-02 至 2026-03-07 |
| 最近 | 过去7天 | 2026-03-01 至 2026-03-07 |

### 搜索规则

1. **每次报告必搜**：
   - `[行业名] + 今日新闻`
   - `[行业名] + 昨日新闻`

2. **禁止使用**：
   - ❌ "近日"
   - ❌ "最近"
   - ❌ "近期"
   - ❌ "一段时间"

3. **必须使用**：
   - ✅ 精确日期：2026-03-07
   - ✅ 相对日期：昨日（2026-03-06）、本周一

## 搜索结果处理

### 信息提取模板

对每条搜索结果，必须提取：

```
{
  "title": "标题",
  "date": "2026-03-07",  // 必填
  "source": "来源",      // 必填
  "summary": "摘要",
  "investment_impact": "利好/利空/中性",
  "verified": true/false
}
```

### 校验清单

每条资讯必须通过：
- [ ] 有明确日期
- [ ] 有明确来源
- [ ] 内容与标题匹配
- [ ] 无明显错误

## 常见问题

### Q: 搜索不到今日新闻怎么办？

A: 
1. 扩大时间范围到"昨日"
2. 搜索"今日" + `[行业名] + 财经`
3. 标记为"暂无今日最新资讯"

### Q: 不同来源日期冲突怎么办？

A:
1. 以权威来源日期为准
2. 标注"时间待确认"
3. 多源交叉验证

### Q: 股价数据找不到怎么办？

A:
1. 使用 Yahoo Finance 官方数据
2. 注明"数据获取失败，待补充"
3. 绝不可编造数据
