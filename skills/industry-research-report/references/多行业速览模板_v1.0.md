# 📊 多行业热点速览模板

> 版本：v1.0 | 创建日期：2026-03-07
> 目标：一页概览今日多行业关键资讯，快速掌握行情走向

---

## 模板结构

```
┌─────────────────────────────────────────────────────┐
│  📊 热点行业速览 - 2026年3月7日                      │
├─────────────────────────────────────────────────────┤
│  今日概览：震荡上行 / 下跌 / 分化                   │
│  核心资讯：X条 | 美股上涨：X个 | A股上涨：X个       │
├─────────────────────────────────────────────────────┤
│  [行业1]  ▶ 详情                                   │
│  [行业2]  ▶ 详情                                   │
│  [行业3]  ▶ 详情                                   │
│  ...                                                │
├─────────────────────────────────────────────────────┤
│  附录：数据更新时间                                  │
└─────────────────────────────────────────────────────┘
```

---

## 汇总页面格式规范

### 1. 顶部概览区

```
# 📊 热点行业速览

**报告日期**：2026-03-07（今日）

### 今日整体判断
- **市场情绪**：[乐观/谨慎/观望]
- **核心观点**：[一句话总结]

### 数据概览
| 指标 | 数量 | 状态 |
|------|------|------|
| 重大资讯 | X条 | ✅ 已更新 |
| 美股上涨 | X个 | ✅ 已更新 |
| A股相关板块上涨 | X个 | ✅ 已更新 |
```

### 2. 行业快速预览卡片

每个行业一行，展示最关键信息：

```
### 量子科技 📈
- **今日关键**：[X条重大资讯]
- **美股对标**：QS (+3.2%) | SLDP (-1.5%)
- **A股板块**：本周 +2.3%
- **趋势**：[上行/下行/震荡] | [详情 →](./quantum_report.html)

---
```

### 3. 资讯汇总（按重要性排序）

按"投资影响"排序：利好 → 中性 → 利空

```
## 今日重大资讯汇总

### 🔴 利好
| 时间 | 行业 | 资讯 | 来源 |
|------|------|------|------|
| 03-07 | 量子科技 | 中国成功研发105比特量子计算机 | 新华社 |

### 🟡 中性
...

### 🔵 利空
...
```

## 数据来源要求

| 数据类型 | 来源 | 更新频率 |
|----------|------|----------|
| 今日重大资讯 | 财联社/同花顺 | 每日 |
| 美股昨日收盘 | Yahoo Finance | 每日 |

```
    │
    ├── 行业1详情 → quantum_report.html
    ├── 行业2详情 → ai_report.html  
    ├── 行业3详情 → new_energy_report.html
    └── ...
```

---

## 页面布局建议

### 样式要求
- 简洁：黑白配色，重点突出
- 快速：一眼看到关键数据
- 跳转：点击行业名或"详情"跳转

### 颜色编码
- 🔴 利好（上涨/突破）
- 🟡 中性（政策/会议）
- 🔵 利空（风险/下跌）

---

## 输出格式

汇总页面输出为 **HTML**，每个行业详情输出为 **独立HTML**。

## 校验清单

```
□ 1. 报告日期是否为"今日"
□ 2. 每条资讯是否有日期
□ 3. 美股是否有涨跌幅数字
□ 4. 行业跳转链接是否有效
□ 5. 是否有一句话趋势总结
```
