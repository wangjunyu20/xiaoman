# 舆情数据获取指南

> 使用东方财富股吧获取市场舆情

## 可用数据源

### 1. 东方财富股吧

**访问方式**：`extract_content_from_websites` 工具

| 数据 | URL | 说明 |
|------|-----|------|
| 个股股吧 | `https://guba.eastmoney.com/list,{代码}.html` | A股代码直接用，港股加 hk 前缀 |
| 行业板块 | `https://guba.eastmoney.com/list,{板块代码}.html` | 需要板块代码 |

### 2. 示例：获取贵州茅台股吧舆情

```python
extract_content_from_websites([{
    "url": "https://guba.eastmoney.com/list,600519.html",
    "prompt": "提取热门帖子标题、评论数、阅读数，按热度排序取前10条"
}])
```

### 3. 行业板块代码参考

| 行业 | 板块代码 |

| 量子科技 | 885626 |

### 4. 在行业报告中使用

生成报告时，可以调用 `extract_content_from_websites` 获取相关股票/板块的股吧舆情，然后综合分析。
