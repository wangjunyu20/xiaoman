# 互联网搜索指南

## 概述

本指南规定如何使用互联网搜索（kimi_search）获取前沿行业资讯，作为新闻抓取的首要步骤。

## 为什么要互联网搜索？

### 传统爬虫的局限
- **前沿科技行业**（量子科技、脑机接口等）新闻源分散
- **最新突破**往往首发于学术期刊（Nature/Science）、公司官网、Twitter/X
- **中文爬虫**难以覆盖英文一手资讯

### 互联网搜索的优势
- 实时获取全球最新动态
- 覆盖学术、产业、投资多维度
- 发现爬虫遗漏的重要新闻

## 搜索策略

### 1. 前沿科技行业搜索模板

#### 量子科技
```
kimi_search("量子科技突破 2025 2026 量子计算 量子通信 国盾量子 本源量子")
```

#### 脑机接口
```
kimi_search("脑机接口突破 2025 2026 Neuralink 马斯克 三博脑科 临床试验")
```

#### 人形机器人
```
kimi_search("人形机器人 2025 2026 特斯拉 Optimus 优必选 量产")
```

#### 可控核聚变
```
kimi_search("核聚变突破 2025 2026 ITER 人造太阳 托卡马克")
```

### 2. 搜索关键词设计原则

| 原则 | 说明 | 示例 |
|------|------|------|
| **时间限定** | 包含年份确保时效性 | `2025 2026` |
| **核心术语** | 行业关键概念 | `量子计算`、`脑机接口` |
| **龙头公司** | 行业代表企业 | `国盾量子`、`Neuralink` |
| **英文原名** | 国际公司/技术用英文 | `Neuralink`、`Optimus` |

### 3. 搜索结果筛选

#### 必须包含的信息
- ✅ 发布时间（确保时效性）
- ✅ 信息来源（权威媒体/机构）
- ✅ 原文链接（可点击跳转）
- ✅ 核心事实（数据/突破/进展）

#### 排除的内容
- ❌ 纯概念炒作文章
- ❌ 没有明确时间/来源的资讯
- ❌ 重复性报道（只保留首发）
- ❌ 纯预测性内容（缺乏事实支撑）

## 搜索结果处理流程

```python
# Step 1: 执行搜索
search_results = kimi_search("量子科技突破 2025 2026")

# Step 2: 解析结果
for item in search_results:
    news = {
        "source": item.source,      # 财联社、新华网等
        "title": item.title,        # 新闻标题
        "url": item.url,            # 原文链接
        "date": item.date,          # 发布时间
        "summary": item.summary,    # 内容摘要
        "impact": analyze_impact(item.title)  # 利好/利空/中性
    }

# Step 3: 与爬虫结果合并（互联网搜索优先）
all_news = merge_news(
    internet_news=search_results,    # 互联网搜索（优先）
    crawler_news=crawler_results,    # 爬虫结果（补充）
    priority="internet_first"        # 去重时保留搜索结果的链接
)

# Step 4: 过滤通用新闻
filtered_news = filter_generic_news(all_news)
```

## 注意事项

1. **搜索配额**：合理使用搜索次数，避免短时间内大量调用
2. **时效性**：优先使用包含年份的关键词，过滤过时内容
3. **交叉验证**：重要新闻需多来源验证，避免单一信源
4. **链接有效性**：确保搜索结果的URL可访问，避免死链
