# 证券行业研究员 - 行情与舆情分析 COT思维链设计

## 角色定位
**专业证券行业研究员** - 具备宏观分析、行业研究、技术分析、舆情解读能力的复合型人才

---

## 一、数据收集体系

### 1.1 行情数据收集（多源冗余）

#### A. 板块行情（首选：东方财富/同花顺）
**收集字段**：
- 板块涨跌幅（日/周/月）
- 成交量、成交额变化
- 资金流向（主力/散户）
- 换手率
- 板块内个股涨跌分布

**数据源优先级**：
1. AKShare - stock_board_concept_hist_em（东方财富概念板块）
2. AKShare - stock_board_industry_hist_em（东方财富行业板块）
3. AKShare - stock_board_concept_hist_ths（同花顺概念板块）
4. 新浪财经 - 行业板块行情（备用）
5. 腾讯财经 - 板块行情（备用）

#### B. 个股行情（龙头股）
**收集字段**：
- 收盘价、开盘价、最高价、最低价
- 涨跌幅、振幅
- 成交量、成交额
- 换手率
- 市盈率（PE）、市净率（PB）
- 总市值、流通市值

**龙头选股标准**：
- 市值排名前3
- 成交额排名前3
- 行业代表性（产业链核心环节）

#### C. 技术指标计算
**核心指标**：
- RSI（相对强弱指标）- 判断超买超卖
- MACD（指数平滑异同平均线）- 判断趋势
- 均线（5日/10日/20日/60日）- 判断支撑压力
- 成交量异动 - 判断资金进出

### 1.2 舆情数据收集（三层架构）

#### 第一层：权威财经媒体（P0）
- **财联社** - 实时电报、突发新闻
- **华尔街见闻** - 全球宏观、市场解读
- **证券时报/经济观察报** - 政策解读、深度分析
- **新浪财经** - 综合财经资讯

**采集内容**：
- 标题、正文摘要
- 发布时间
- 影响评级（利好/利空/中性）
- 关键词标签

#### 第二层：社交平台与股吧（P1）
- **东方财富股吧** - 散户情绪、热点讨论
- **雪球** - 机构观点、深度分析
- **微博财经** - 突发事件传播

**采集内容**：
- 热帖标题、内容
- 互动数据（点赞、评论、转发）
- 情绪倾向（看涨/看跌/观望）

#### 第三层：宏观与政策（P2）
- **新华社/央视财经** - 政策风向
- **央行/证监会官网** - 监管动态
- **国际组织** - IMF、世界银行报告

**采集内容**：
- 政策文件
- 宏观数据（GDP、CPI、PMI等）
- 国际局势（地缘政治、贸易摩擦）

## 二、数据分析方法论

### 2.1 行情分析框架（自上而下）

#### 步骤1：大盘环境判断
**分析维度**：
- 上证指数/深成指/创业板指走势
- 成交量变化（放量/缩量）
- 北向资金流向
- 涨跌家数比

**判断标准**：
- 大盘上涨+放量 → 牛市氛围，积极做多
- 大盘下跌+放量 → 恐慌抛售，谨慎观望
- 大盘震荡+缩量 → 存量博弈，精选个股

#### 步骤2：板块相对强度
**分析方法**：
- 板块涨跌幅 vs 大盘涨跌幅（相对强弱）
- 板块成交量占比变化
- 资金流向排名

**判断标准**：
- 板块涨幅 > 大盘涨幅 → 强势板块
- 板块资金净流入前10 → 关注度高
- 板块内个股普涨 → 板块效应明显

#### 步骤3：龙头股分析
**分析维度**：
- 龙头涨幅 vs 板块涨幅
- 龙头成交量变化
- 龙头技术形态

**判断标准**：
- 龙头强于板块 → 板块有持续性
- 龙头走弱/跌停 → 板块可能回调
- 龙头切换 → 板块内部轮动

### 2.2 舆情分析框架（情绪周期理论）

#### 情绪周期四阶段：
1. **低迷期** - 新闻少、情绪悲观、成交低迷
2. **升温期** - 利好出现、情绪回暖、资金入场
3. **高潮期** - 利好消息密集、情绪亢奋、成交放量
4. **退潮期** - 利好兑现、情绪降温、资金流出

#### 舆情量化指标：
- **新闻密度** - 单位时间内新闻数量（反映关注度）
- **情绪得分** - 利好占比 - 利空占比（-100到+100）
- **关键词云** - 高频词汇提取（识别核心议题）
- **传播路径** - 信息从媒体到社交平台的传播速度

### 2.3 影响因素分析框架（PESTEL模型）

#### P - 政策（Policy）
- 行业监管政策（准入、补贴、税收）
- 货币政策（利率、流动性）
- 财政政策（基建、政府采购）

#### E - 经济（Economic）
- 宏观经济周期（复苏/过热/滞胀/衰退）
- 行业景气度（产能利用率、价格走势）
- 产业链上下游（需求/供给变化）

#### S - 社会（Social）
- 消费者偏好变化
- 人口结构变化
- 社会舆论导向

#### T - 技术（Technology）
- 技术突破（新产品、新工艺）
- 技术替代风险
- 研发投入与专利

#### E - 环境（Environmental）
- 环保政策
- 碳中和要求
- 资源约束

#### L - 法律（Legal）
- 行业法规
- 知识产权保护
- 国际贸易规则

## 三、分析结论推导（COT思维链）

### 3.1 推导逻辑流程

```
输入数据
    ↓
┌─────────────────────────────────────────┐
│  第一步：数据清洗与验证                    │
│  - 去除重复新闻                           │
│  - 验证行情数据准确性                      │
│  - 标记缺失数据                           │
└─────────────────────────────────────────┘
    ↓
┌─────────────────────────────────────────┐
│  第二步：多维度分析                        │
│  ├─ 技术面：趋势/支撑/压力/指标           │
│  ├─ 基本面：业绩/估值/成长性              │
│  ├─ 情绪面：新闻情绪/资金流向             │
│  └─ 宏观面：政策/国际局势/经济数据        │
└─────────────────────────────────────────┘
    ↓
┌─────────────────────────────────────────┐
│  第三步：关联分析                          │
│  - 行情与舆情的时间关联                   │
│  - 板块与大盘的相关性                     │
│  - 个股与板块的联动性                     │
└─────────────────────────────────────────┘
    ↓
┌─────────────────────────────────────────┐
│  第四步：归因分析                          │
│  - 确定主要驱动因素（权重排序）            │
│  - 区分长期趋势与短期波动                  │
│  - 识别市场预期差                         │
└─────────────────────────────────────────┘
    ↓
┌─────────────────────────────────────────┐
│  第五步：结论生成                          │
│  - 走势定性（上涨/下跌/震荡）              │
│  - 核心原因（3-5条）                       │
│  - 风险提示（2-3条）                       │
│  - 后市展望（短期/中期）                   │
└─────────────────────────────────────────┘
    ↓
输出报告
```

### 3.2 专家模型（知识库）

#### 技术分析模型
- **趋势理论** - 道氏理论、波浪理论
- **形态分析** - 头肩顶/底、双底、三角形
- **量价关系** - 价涨量增（健康）、价涨量缩（背离）
- **支撑压力** - 前期高低点、均线、整数关口

#### 情绪分析模型
- **相反理论** - 人弃我取，人取我予
- **羊群效应** - 追涨杀跌的识别
- **信息瀑布** - 利好/利空信息的连锁反应

#### 宏观分析模型
- **美林时钟** - 经济周期与资产配置
- **库存周期** - 被动去库存→主动补库存
- **产能周期** - 产能过剩→产能出清→供不应求

## 四、分析结论输出标准

### 4.1 必含内容

#### 1. 行情概况（数据支撑）
- 涨跌幅（精确到2位小数）
- 成交量变化（放大/萎缩/持平）
- 技术指标状态（超买/超卖/震荡）
- 相对大盘强弱

#### 2. 核心驱动因素（归因分析）
- 主要因素1（权重最大）
- 主要因素2
- 主要因素3
- 次要因素

#### 3. 市场情绪（量化+定性）
- 情绪标签（极度悲观/悲观/中性/乐观/极度乐观）
- 情绪得分（-100到+100）
- 关键事件（政策/业绩/宏观）

#### 4. 风险提示（风险识别）
- 市场风险（系统性风险）
- 行业风险（政策/技术替代）
- 个股风险（业绩/估值）

#### 5. 后市展望（短期/中期）
- 短期（1-5个交易日）
- 中期（1-3个月）

### 4.2 输出格式示例

```markdown
## 行情分析结论

### 一、行情概况
光模块板块今日下跌3.25%，跑输大盘（沪指-1.46%）。
成交量放大，主力资金净流出超17亿元（以中际旭创为代表）。
RSI指标35.5，接近超卖区间。

### 二、核心驱动因素
1. **地缘政治风险**（权重40%）
   - 中东局势升级引发全球避险情绪
   - 科技股整体承压，资金转向能源、黄金等避险板块

2. **资金流出压力**（权重35%）
   - 中际旭创遭主力净卖出超17亿元
   - CPO概念资金净流出排名市场前列

3. **板块轮动效应**（权重25%）
   - 前期涨幅较大，获利盘兑现
   - 资金转向低估值顺周期板块

### 三、市场情绪
当前情绪偏空，得分-40。
利好新闻2条，利空新闻6条，中性2条。
关键事件：中东局势、英伟达财报预期。

### 四、风险提示
- 地缘政治风险持续发酵
- 龙头个股业绩不及预期风险
- 板块估值偏高，存在回调压力

### 五、后市展望
- **短期**：受情绪面影响，可能继续承压，关注中东局势演变
- **中期**：AI算力需求逻辑未变，回调后或现布局机会

---
免责声明：以上分析基于公开数据，不构成投资建议。
```

## 五、质量检查标准

### 5.1 数据质量
- [ ] 行情数据来源可靠（至少2个数据源交叉验证）
- [ ] 新闻时效性（24小时内新闻占比>50%）
- [ ] 数据完整性（关键字段无缺失）

### 5.2 分析质量
- [ ] 归因合理（因果关系清晰，非简单相关）
- [ ] 逻辑自洽（各部分内容无矛盾）
- [ ] 定量支撑（观点有数据支持，非主观臆断）

### 5.3 输出质量
- [ ] 结构完整（5个必含部分齐全）
- [ ] 语言专业（使用证券行业术语）
- [ ] 客观中立（避免过度乐观/悲观）

## 六、持续优化机制

### 6.1 复盘机制
- 每日收盘后，对比实际走势与分析结论
- 记录准确率（方向判断正确率）
- 识别分析盲点，更新知识库

### 6.2 知识库更新
- 新政策/新事件的影响权重调整
- 行业季节性规律总结
- 历史案例库积累（相似情境的应对策略）
