# 因果关系原理详解

## 链条1: 通胀预期上升 → 美联储维持高利率预期

### 核心原理: 泰勒规则 (Taylor Rule)

美联储的货币政策目标:
1. **充分就业** (失业率目标)
2. **物价稳定** (通胀目标 2%)

### 传导机制

```
霍尔木兹海峡封锁
    ↓
石油供应减少 20%
    ↓
油价暴涨 (70→108美元, +54%)
    ↓
【成本推动型通胀】
- 能源成本占CPI权重约7%
- 油价上涨→运输成本上涨→商品价格上涨
- 通胀预期从2%上升至3-4%
    ↓
【美联储反应函数】
根据泰勒规则: 名义利率 = 均衡利率 + 1.5×(通胀-目标) + 0.5×(产出缺口)

当实际通胀 > 目标通胀(2%)时:
- 美联储有义务提高利率以抑制通胀
- 市场预期美联储将维持高利率更长时间
- 甚至可能需要进一步加息
    ↓
【结果】
- 市场预期降息时间推迟
- 终端利率预期上调
- 高利率预期强化
```

### 关键数据
- **当前油价**: 108美元/桶 (vs 危机前70美元)
- **通胀预期**: 从2%上升至3.5%+
- **美联储目标**: 2%通胀
- **预期利率路径**: 2026年全年维持4.5-5.0% (vs 之前预期的3.5-4.0%)

## 链条2: 美联储维持高利率预期 → 科技股估值承压

### 核心原理: 贴现现金流模型 (DCF Valuation)

股票价值 = 未来现金流的现值

```
价值 = Σ [CF_t / (1+r)^t]

其中:
- CF_t = 第t期的现金流
- r = 贴现率 (无风险利率 + 风险溢价)
```

### 传导机制

```
【高利率环境的直接影响】
美联储维持高利率 (4.5-5.0%)
    ↓
无风险利率上升 (10年期美债收益率上升)
    ↓
贴现率 r 上升
    ↓
【DCF模型计算】
未来现金流的现值 = CF / (1+r)^t

当 r 上升时:
- 分母变大
- 现值下降
- 股票估值下降

特别是科技股:
- 科技股现金流主要在未来 (成长性)
- 远期现金流折现因子: 1/(1+r)^t
- t越大，r上升影响越大
    ↓
【具体例子】
假设某AI公司:
- 今年现金流: 10亿
- 5年后现金流: 100亿
- 10年后现金流: 300亿

当贴现率从8%上升到12%:
- 今年现金流现值: 10→9.3 (降7%)
- 5年后现金流现值: 100→56.7 (降43%)
- 10年后现金流现值: 300→96.6 (降68%)

→ 科技股估值大幅压缩
```

### 第二个机制: 资金流向

```
高利率环境
    ↓
债券收益率上升 (国债4-5%)
    ↓
【资产配置转移】
投资者对比:
- 科技股: 高风险，预期收益10-15%，但不确定性高
- 国债: 无风险，确定收益4-5%
    ↓
风险偏好下降
    ↓
资金从科技股流向债券/现金
    ↓
科技股卖压增大
```

### 第三个机制: AI投资预期

```
高利率 → 美国财政融资成本上升
    ↓
美债销售困难 → 财政赤字压力
    ↓
政府支出受限
    ↓
AI产业补贴/投资减少预期
    ↓
AI公司收入预期下调
    ↓
科技股估值进一步下调
```

    ↓ (行业关联)

## 关键结论

### 为什么CPO跌得特别多？

1. **高估值**: AI板块前期涨幅大，估值处于高位
2. **远期现金流占比高**: 成长性公司现金流主要在远期，对利率更敏感
3. **资金集中**: 筹码结构过于集中，获利盘丰厚，对利空敏感
4. **产业关联**: AI投资预期直接影响光模块需求预期

### 如果局势缓解，会怎么走？

**反向链条**:
```
霍尔木兹海峡重新开放
    ↓
油价回落
    ↓
通胀预期下降
    ↓
美联储降息预期提前
    ↓
贴现率下降
    ↓
科技股估值修复
    ↓
CPO/光模块反弹
```

### 风险提示

1. **地缘风险**: 如果封锁持续，油价可能冲至150美元
2. **通胀失控**: 如果美联储控制不住通胀，可能被迫大幅加息
3. **经济衰退**: 高利率可能导致经济硬着陆，科技股盈利下调
4. **AI泡沫**: 如果AI需求不及预期，估值可能进一步压缩
